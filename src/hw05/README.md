## JavaScript 陣列(Array)也有自己的過濾函數。

- 請參考 `get_gt10(...)` 與 `get_has3(...)` 的設計方式，另外撰寫『質數』過濾函數。
    - `get_gt10(...)` 使用的是箭頭(=>)函數(arrow function)定義方式。
    - `get_has3(...)` 當要定義的處理方式較複雜，則會採用外部函數的定義方式。

### 在 Windows 簡易測試
```shell
lab05> 1..30 | Get-Random -Count 20 | node .\main.js

lab05> 1..30 | Get-Random -Count 20 | node .\main.js

lab05> 1..30 | Get-Random -Count 20 | node .\main.js

```
